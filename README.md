# 📖 About

AI Virtual Mouse is a beginner-friendly Python project that demonstrates how Artificial Intelligence and Computer Vision can control a computer mouse using only hand gestures.

The project uses **OpenCV** for webcam access, **MediaPipe** for real-time hand tracking, and **PyAutoGUI** for mouse automation.

With just a webcam, you can move the cursor, perform mouse clicks, double-click, and scroll using simple hand gestures.

This project is perfect for anyone interested in learning:

- Python
- Artificial Intelligence
- Computer Vision
- OpenCV
- MediaPipe
- Hand Tracking
- Mouse Automation
- Human Computer Interaction (HCI)

# 🎯 Gesture Controls

| Gesture | Action |
|----------|--------|
| ☝️ Index Finger | Move Cursor |
| 🤏 Thumb + Index Finger | Left Click |
| 🤏 Thumb + Middle Finger | Right Click |
| 🤏 Thumb + Ring Finger | Double Click |
| ⬆️ Move Finger to Top | Scroll Up |
| ⬇️ Move Finger to Bottom | Scroll Down |

# 🔄 Uninstall TensorFlow (If Needed)

If you have **TensorFlow** installed and encounter **MediaPipe**, **NumPy**, or **Protobuf** compatibility issues, uninstall TensorFlow before running this project.

Run:

```bash
py -3.12 -m pip uninstall tensorflow tensorflow-intel keras
```

After uninstalling, reinstall the required packages:

```bash
pip install opencv-python mediapipe pyautogui
```

> **Note:** This step is only required if you have TensorFlow installed. If you don't use TensorFlow, you can safely skip it.

# 🖱️ How to Use

Start the program.

Allow webcam access if prompted.

Place your hand inside the purple tracking box.

Control your mouse using the following gestures.

---

## ☝️ Move Cursor

Move your **Index Finger** to control the mouse cursor.

---

## 👆 Left Click

Pinch your **Thumb** and **Index Finger** together.

---

## 🖱️ Right Click

Pinch your **Thumb** and **Middle Finger** together.

---

## ⚡ Double Click

Pinch your **Thumb** and **Ring Finger** together.

---

## 📜 Scroll Up

Move your **Index Finger** near the **top** of the tracking area.

---

## 📜 Scroll Down

Move your **Index Finger** near the **bottom** of the tracking area.
